# LEITORES CLIENT

Controle de avaliação de livros. Este sistema tem como objetivo fazer um levantamento das opniões dos alunos, relativo aos livros que efetuou a leitura, informando o estado de conservação do livro, nota e observação - CLIENT

## Iniciar o projeto

- Baixar repositório

- Modo desenvolvimento: 
```bash
npm install
```
```bash
ng serve
```

## Tecnologias utilizadas

  * [TypeScript](https://www.typescriptlang.org/)
  * [Angular 5](https://angular.io/)
  * [JWT](https://jwt.io/)
  * [Express](http://expressjs.com/pt-br/)
  * [Angular Material](https://material.angular.io/)
